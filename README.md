## how to install

### .env file

```
DATABASE=./data
```

## how to use

### Buy 1 Tether (USDT) in Brazilian Real (BRL)

```
bash ./src/markets.quotes.sh buy 1 usdt brl 
```

### Get BTCBRL "1 day candles" by 1 week ago

```
bash ./src/markets.candles.sh btcbrl 1d 1week 
```
